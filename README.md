# Scam:Vouch Discord Bot

A user-friendly Discord bot that helps track user reputation through scam and vouch reports. Easily check user histories and see their reputation status at a glance!

## Features

- **Reputation Tracking**: Track scams and vouches per user
- **Automatic Roles**: Roles are automatically assigned based on reputation counts
- **Search Functionality**: Search for any user's reputation in the vouch channel
- **Easy to Use**: Simple slash commands for everyone

## Commands for Everyone

### Basic Commands

- **`/scam @user [reason]`** - Report a user as a scammer
  - Example: `/scam @JohnDoe He scammed me on a trade`

- **`/vouch @user [reason]`** - Vouch for a user
  - Example: `/vouch @JaneDoe Great trader, very trustworthy`

- **`/user-check @user`** - Check a user's scam and vouch counts
  - Example: `/user-check @JohnDoe`
  - Shows: Scam count, Vouch count, and current reputation roles

### Search Command (Vouch Channel)

- **`/search @user`** - Search for detailed reputation information about a user
  - Works best in the vouch channel (if configured)
  - Shows: Detailed stats, recent reports, and reputation roles
  - Example: `/search @JohnDoe`

## How It Works

1. **Reporting**: Use `/scam` or `/vouch` to report on a user's behavior
2. **Automatic Roles**: The bot automatically assigns roles like "Scams 3" or "Vouches 5" based on counts
3. **Checking**: Use `/user-check` or `/search` to see any user's reputation
4. **Roles**: Users receive colored roles that show their reputation at a glance

## Understanding Reputation

- **Scam Count**: Number of scam reports received
- **Vouch Count**: Number of positive vouches received
- **Reputation Roles**: Automatically assigned roles like "Scams 4" or "Vouches 10"
  - Scam roles are red/orange colored
  - Vouch roles are green/teal colored

## Tips

- Always provide a reason when reporting - it helps others understand the situation
- Use `/user-check` for quick reputation looks
- Use `/search` in the vouch channel for detailed information
- Check someone's reputation before trading or doing business with them

## Need Help?

- Staff members can remove incorrect reports
- If you see an error, let staff know!
